# Realistische BOS-Funk Transkripte (Feuerwehr Bayern, Digitalfunk)

Orientiert an DV 810 / Sprechfunk-Regeln

## Typische Merkmale

- Aufbau: "Empfänger von Sender kommen"
- Antworten mit: "Hier … kommen"
- Abschluss ohne "kommen", wenn Gespräch beendet → letztes "Verstanden" OHNE "kommen"
- Kurz, klar, keine Umgangssprache
- Zahlen einzeln gesprochen ("zwo")

## 1. Standard: Fahrzeug ↔ Leitstelle

```
Florian Coburg 40/1 von Leitstelle Coburg kommen

Hier Florian Coburg 40/1 kommen

Florian Coburg 40/1 Einsatz für Sie, Brand B3, Wohnhausbrand,
Coburg, Musterstraße 12, kommen

Verstanden, B3 Wohnhausbrand Musterstraße 12,
Florian Coburg 40/1 rückt aus, kommen

Verstanden
```

## 2. Lagemeldung an Leitstelle

```
Leitstelle Coburg von Florian Coburg 40/1 kommen

Hier Leitstelle Coburg kommen

Lage auf Sicht: starker Rauch aus 1. Obergeschoss,
Menschenleben in Gefahr, wir gehen mit einem Trupp unter Atemschutz vor,
kommen

Verstanden, weitere Kräfte werden alarmiert, kommen

Verstanden
```

## 3. Nachforderung (z.B. Wasser / Tanklöschfahrzeug)

```
Leitstelle Coburg von Florian Coburg 40/1 kommen

Hier Leitstelle Coburg kommen

Wir benötigen zusätzlich ein Tanklöschfahrzeug,
Wasserversorgung unzureichend, kommen

Verstanden, Florian Coburg 21/1 wird nachalarmiert, kommen

Verstanden
```

## 4. Einsatzstellenfunk (DMO, intern)

```
Angriffstrupp von Florian Coburg 40/1 kommen

Hier Angriffstrupp kommen

Wie ist eure Lage, kommen

Wohnung stark verraucht, keine Person gefunden,
gehen weiter Richtung Schlafzimmer vor, kommen

Verstanden, Rückmeldung bei Person oder Brandherd, kommen

Verstanden
```

## 5. Statusmeldung + kurze Kommunikation

```
Leitstelle Coburg von Florian Coburg 40/1 kommen

Hier Leitstelle Coburg kommen

Sind an der Einsatzstelle eingetroffen, kommen

Verstanden
```

## 6. Einsatzende

```
Leitstelle Coburg von Florian Coburg 40/1 kommen

Hier Leitstelle Coburg kommen

Feuer aus, Nachlöscharbeiten laufen,
keine weiteren Kräfte erforderlich, kommen

Verstanden
```
